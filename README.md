# SmilingPanda Starter Kit

## Hi everybody

Here is my simply react/redux starter with few technologies.

* Typescript
* React
* React Router
* Redux + replace reducers
* Webpack
* Babel
* SASS

### install

```
git clone https://github.com/akopichin/webpack-react-template
cd webpack-react-template/
npm i
npm run build
// to run the server
npm run start
```
